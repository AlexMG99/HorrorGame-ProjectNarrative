## Intro
Forest Escape is a first persone horror/puzzle game made by Alex Morales, Guillem Sánchez and Adria Perez for Realtime Narrative subject. You can 
check a look to the game.

## Controles:
- WASD: Movimiento del personaje
- E: Interactuar
- Space: Close Eyes
- Esc: Cerrar juego
